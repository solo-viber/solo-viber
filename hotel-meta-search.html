Description

This project is a hotel meta-search engine similar to Trivago. It will allow users to:

  • Search hotels by destination, dates, and number of guests
  • Compare prices from multiple booking platforms
  • Filter by amenities, star rating, and distance
  • View hotel details and redirect to partner websites for bookings

Technology Stack

  • Frontend: Next.js, Tailwind CSS
  • Backend: Node.js (Express) / Serverless functions
  • APIs: Travelpayouts Hotellook API, Hotels Data API
  • Deployment: Planned on AWS

Monetization Strategy

The platform will use affiliate links and partner redirection through Travelpayouts. Users who click through and make bookings will generate commission revenue.

Development Timeline

  • Current Phase: API integration and UI prototyping
  • Planned MVP Launch: August or September 2025
  • Next Phase: Advanced filters, SEO, mobile responsiveness
